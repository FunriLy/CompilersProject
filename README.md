- 实验环境与工具 

  - 计算机及操作系统： PC 机， Windows10
  - 实现工具： C++Builder6
  - 教学型编译程序： PL/0    

- 课程设计内容

  - 增加单词：保留字 ELSE， FOR， STEP， UNTIL， DO， RETURN 

  - 增加单词：运算符 *=， /=， &， ||，！ 

  - 修改单词：不等号# 改为 <> 或 !=    

  - 增加条件语句的 ELSE 子句

  - 扩充赋值运算： *= 和 /=    

  - 扩充语句（Pascal 的 FOR 语句） : 

    **FOR <变量>:=<表达式>STEP<表达式> UNTIL<表达式>Do<语句>    **

  - 扩充运算： ++ 和 --    

  - 增加 // 单行注释和 /* */ 多行注释
